# DataSQRL Workshop Materials

This repository contains the following materials for the DataSQRL "Data Engineering Automation" workshop:

1. Data Catalog: Connector definitions for the core data assets of our fictional organization
   1. [E-Commerce Sources](ecommerce-sources/README.md): Defines the core data sources for an E-Commerce platform (customers, sellers, products, orders, fulfillment, etc)
2. [Sample Implementation](sample-implementation): Sample data product implementation with DataSQRL using the E-Commerce dataset.
